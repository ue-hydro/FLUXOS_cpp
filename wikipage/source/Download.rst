Download
==================================

FLUXOS is hosted on GitHub. Clone the repository — submodules are not used, so
a plain ``git clone`` is sufficient:

.. code-block:: bash

   git clone https://github.com/ue-hydro/FLUXOS_cpp.git
   cd FLUXOS_cpp

Once inside the checkout, proceed to :doc:`Containers` for the recommended
container-based build path (Docker on laptops, Apptainer on HPC) or to the
OS-specific native build guides (:doc:`Linux`, :doc:`MacOS`, :doc:`Windows`).
