Git and GitHub
==================================

The FLUXOS repository is hosted at
https://github.com/ue-hydro/FLUXOS_cpp.

Developers should fork ``master`` to carry out new developments, then open a
pull request against the upstream ``master``. Before starting a substantial
change, get in touch with the maintainer (diogo.costa@uevora.pt) so we can
coordinate and make sure the starting point of your branch is up to date.

For an overview of the project layout and quick-start instructions, see the
top-level ``README.md`` or https://github.com/ue-hydro/FLUXOS_cpp#readme.
