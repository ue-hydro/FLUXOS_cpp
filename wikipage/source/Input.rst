FLUXOS I/O
==================================

FLUXOS reads a single ``modset.json`` configuration file that points at
the DEM, optional mesh, forcing files, and output settings — all of which
are normally produced in one shot by the ``model_config_template.py``
template (see :doc:`SupportingScripts`). This section documents the input
and output *formats* themselves so you can inspect, hand-edit, or consume
them directly.
